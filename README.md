# Shentry 

```
Sentry — это платформа для отслеживания ошибок и
мониторинга приложений.
Данные, которые мы получаем в Sentry,
содержат исчерпывающую информацию о контексте,
в котором это произошло, что облегчает воспроизведение и поиск ...
```

Шентри, это дурацкое мини приложение - замена сентри, для маленьких проектов совместимое с sentry_sdk
Sentry - это промышленный сервис, для сбора ошибок и/или логов


Оно специально такое тупое и испольует sqlite, чтобы минимизировать сложность развертывания
